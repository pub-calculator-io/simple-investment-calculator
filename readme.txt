Free investment calculator that uses the formula PV (1 + R)^n to help investors understand how to calculate investment returns and analyze investments.

[https://www.calculator.io/simple-investment-calculator/](https://www.calculator.io/simple-investment-calculator/)

== Usage ==

Add the Simple Investment Calculator shortcode to your page, post or sidebar:

`[ci_simple_investment_calculator]`

Add the following code to your template where you would like the Simple Investment Calculator to appear:

`<?php display_ci_simple_investment_calculator(); ?>`

== Installation ==

1. Upload the Simple Investment Calculator /ci_simple_investment_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Simple Investment Calculator plugin through the "Plugins" menu in WordPress.
